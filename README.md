## Usage
1. Navigate into the project directory:
    ```bash
    cd slime-rancher-2-interactive-map
    ```
2. Install the required packages:
    ```bash
    npm install
    ```
3. Run the web app:
    ```
    npm start
    ```
    
## Current features:
- See locations for:
  - Gordo Slimes
  - Bee Drones
  - Map Nodes
  - 7Zee Treasure Pods
- Place custom icons
- Save custom icons to browser
